# Changelog

### v0.3.0 Alpha | 2025-11-30
1. 经历了一次大的重构，将fastapi引入项目。
2. 使用 module-role 架构，将项目分为多个模块，每个模块负责不同的功能。
3. 实现了一个通过json动态生成配置项的 web 界面，用于项目启动参数的配置和运行中的交互。
4. 这是第一个完整上传到 GitHub 项目（SwarmClone/SwarmClone）的版本。

### v0.2.0 Alpha | 2024-12-25
1. 经历了一次大的重构。

2. 将多进程架构改为多线程架构，websocket通信。

3. 实现了基础的LLM、TTS、ASR交互功能。

### v0.1.0 Alpha | 2024-12-10
1. 第一个可运行的版本，使用多进程架构，websocket通信。

2. 由于很早就被废弃，相关代码已经无从考证。
